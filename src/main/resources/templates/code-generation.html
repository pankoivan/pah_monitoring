Описание

При нажатии на кнопку "Сгенерировать" сервер создаёт уникальный код: пароль для доступа к странице регистрации медицинского учреждения.

Не отправляйте сгенерированный код посторонним людям. Он предназначен только для официальных представителей медицинских учреждений, подтвердивших свою личность и намерение использовать приложение.

В единственном поле "Срок действия кода" выбирается число дней, которое код будет храниться в базе данных. После успешной регистрации медицинского учреждения код удаляется из базы. Если срок действия кода истёк, а медицинское учреждение по каким-то причинам не было зарегистрировано, генерацию и выдачу кода нужно будет провести заново.
